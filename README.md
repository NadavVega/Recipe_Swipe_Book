# Recipe Swipe Book üç≥‚ú®

A beautifully crafted recipe discovery app designed to match your energy. Feeling lazy? Enter **Quick Meal** mode and swipe your way to the perfect dish. Hosting friends? Switch to **Build a Feast** to curate a flawless, multi-course menu with a smart, cross-referenced cuisine filter.

Complete with smooth micro-interactions, an airy aesthetic UI, and local storage to keep your favorite menus saved. No frameworks, just pure, elegant Vanilla JS.

## ‚ú® Features

- **Dual Modes:** Swipe for quick weeknight meals or build a structured 4-course feast.
- **Smart Filtering Algorithms:** Dynamically calculates and displays only the valid cuisines for your selected meal category.
- **Local Persistence:** Saves your liked recipes and fully planned feasts using `localStorage`.
- **Shareable Menus:** Uses the native Web Share API to easily send your curated feasts to friends.
- **Zero Dependencies:** Built entirely with Vanilla HTML, CSS, and JS. No complex build steps required.
